git init - создание репозитория
git status - статус репозитория
git add - добавить файлы в индекс
git commit - сделать коммит
git commit -am работает только для файлов в репозитории, новые файлы опция игнорирует
git diff - разница между рабочей дирректорией и репозиторием
git diff --staged - разница между индексом и ропозиторием
